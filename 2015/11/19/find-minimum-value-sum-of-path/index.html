Find Minimum Value Sum of Path

2015-11-19 · 63 words · 1 minute read
Leetcode YMX Algorithm
Binary Tree

Given a binary tree, find a path has the minimum sum of node value.

Solution

public class Solution{
	public static int minPath(TreeNode root) {
		if (root == null)
			return 0;
		if (root.left == null && root.right == null)
			return root.val;
		if (root.left == null)
			return root.val + minPath(root.right);
		if (root.right == null)
			return root.val + minPath(root.left);
		return root.val + Math.min(minPath(root.left), minPath(root.right));
	}
}
